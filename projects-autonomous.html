Autonomous Driving Vehicle

Introduction

This project started in the academic year 2223S1 and it consists of a team of 4 members from the Diploma of Mechatronics & Robotics (DMRO) in SP, the engineering skill sets used in this project are mainly electrical, mechanical and programming

Objective

The objective of this project is to achieve wireless control of the vehicle's DC Brushless Motor with computer vision, as well as to demonstrate real time monitoring and control from a graphical user interface (GUI)

Scope

For the computer vision part, the team decided to use neural network to detect the different traffic light states, red and green to control the motors of the vehicle.

Key components

Mechanical

  • Driving wheels are brushless DC motors
  • Steering mechanism

Electrical

  • Odrive (motor controller)
  • Nvidia Jetson Nano
  • Raspberry Pi 4B
  • Arduino

Programming

  • Graphical User Interface (GUI)
  • Proximity Detection

Artificial Intelligence

  • Traffic lights detection
  • Lane detection

Electrical Components

Odrive motor controller

This motor controller is powered by a battery, is used to control the driving DC brushless motors of the vehicle, it also has a brake resistor.

Nvidia Jetson Nano

Nvidia Jetson Nano is a key component as it programs the Odrive motor controller of the vehicle which in turns drives the DC brushless motors.

Raspberry Pi 4B

Raspberry Pi 4B is another key component as it operates the ultrasonic sensors as well as control the Arduino.

Arduino

Arduino is the last key component as it operates the temperature sensors as well as the steering of the vehicle

Mechanical Components

Brushless DC motors

The driving motors are controlled using the onboard Odrive motor controller which is capable of controlling the motors speed with precision.

Steering Mechanism

The steering mechanism used is ackermann steering, the motors are driven by a motor driver. Feedback on turn angle is done with potentiometers and discrete ADCs. Calibration of the steering motor is done to get a reference of the steering angles and to determine the extreme limits of the linear actuator.

Programming

Graphical user Interface (GUI)

The graphical user interface is created using Tkinter, a python library used to create interfaces. In this interface, the user can monitor the speed, temperature at crucial parts of the vehicle, voltage of the battery as well as any system errors. The user can also control the vehicle from the interface by either using the mouse or pressing the "WASD" keys on the keyboard.

Proximity Detection

By reading data from the ultrasonic sensor, the program is able to stop the vehicle when the distance is within close proximity.
